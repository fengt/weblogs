你不知的心声

发表于 2016-09-10 | 分类于 随笔

总认为自己的心声很多时候不为人知没有什么，因为彼此本不熟悉，也不了解。但时间愈久，便会发觉曾经熟悉的人或事已渐行渐远，若隐若现。

近来也是对闲暇琐碎时间里浏览了很多微信公众号的文章，有很多体会、收获和成长：或某段文字以亲近共鸣，或某句话以醍醐灌顶，或谈及某本书以受益匪浅，或身临其境倍感一股动力…感慨体味百般，让人欢喜，有时愁！原来，世界很小，彼此内心在靠近，志趣相投的人走在一起。如此，便能欢呼，便能呐喊，便能被予理解，便能真我长现！

时常寄以文字来真挚表达内在的心声，继而可梳理和反思自己。不同的是，没有勇气或是其他合适的理由表露这些心声于外界，显得那么狭小和不可理解，给身边的人留下困惑，也让自己难以释怀。

我想，现在开启新的篇章，鼓起勇气，去开放心态，寻找内心的宁静！

事务浅识

发表于 2015-11-28 | 分类于 技能

1. Spring定义事务属性几种方式

  • 事务的传播行为(Propagation Behavisor)

    • required: 支持当前事务，如果当前没有事务，就新建一个事务（默认）

    • supports: 支持当前事务，如果当前没有事务，就以非事务方式执行

    • mandatory: 支持当前事务，如果当前没有事务，就抛出异常

    • requires_new: 新建事务，如果当前存在事务，把当前事务挂起

    • not_supported: 以非事务方式执行操作，如果当前存在事务，就把当前事务挂起

    • never: 以非事务方式执行，如果当前存在事务，则抛出异常

    • nested: 如果当前存在事务，则在嵌套事务内执行。如果当前没有事务，则执行与propagation_required类似的操作

  • 事务的隔离级别(Isolation)

  • 事务的超时时间(Timeout)

  • 是否为只读事务(ReadOnly)

星空，六月末的惊喜

发表于 2015-06-30 | 分类于 随笔

杨柳青青江水平，闻郎岸上踏歌声。
东边日出西边雨，道是无晴却有晴。

已经不记得多久没有见到像今晚的星空：皎洁的月光、闪闪发亮的星星。激动、激动、除了激动还是激动……

小的时候，小的时候，满脑子回忆起过去的真切、欢乐和忧愁……

现在，却很少有这种感受，心随时间慢慢销蚀，感触、灵感、会心、欢笑，同学、朋友、伙伴、亲人……这些，似乎越来越远，伴随着步伐稀稀落落。也许工作太忙，也许欲望太多，也许停不了脚步，也许已无选择，太多的因素促成，也不再回落。向前走，慢慢来，时间会让我们变得更好！

人间情义难易懂，唯有景色撩人心。爱，从何而来？情，又该如何抑扬？六月，六月，六月是一个充满惊喜的时间点，不管世间此起彼伏，你一如既往偷偷地漏出了色采，意者无意发现了你，而你却把他带回了童年的记忆……惊喜，不得不说的惊喜！还记得以往的初心、儿时的欢喜、过去的青春岁月吗？那是一个想回却再也回不去的年代，看看来时的路，想想每每的感受，心在哪？梦在哪？

星空，星空！你承载了太多往昔的痕迹，太多伙伴们一起欢快的畅想，你不辜负意者的期望，却又来的这么巧合！你如同一颗会心的xx（什么词好呢），滋润着我们生长的每寸土地，散发着柔和、温馨而不乏力的光芒……

压力这玩意儿

发表于 2015-06-27 | 分类于 随笔

生于忧患，死于安乐。

心急是一种病，得治！压力是一种心理战术，需冷静和睿智！

压力，其实来自于自己想尽短时间内解决问题，但往往是相反，因为既然产生压力，便不会很容易就得到解决，也不能心急一股劲想完成，做事一旦心急，便不能长期坚持做好。

正确的方法：压力分解。分析形成压力的原因，将难以解决的问题进行分解，逐步解决每个环节，过度一个时间段，便会发现这些环节串在一起就是之前形成的压力，也便是消除压力的时间点。其间，环节的分解和时间段的过度及把握是关键！这将决定自己的能力和解决问题的效率，值得注意的应是这两点的学习、积累和不断的坚持。

无聊之想，仅作警示。

这儿的纠结

发表于 2015-06-14 | 分类于 随笔

最近工作很忙，整个人好像也停止了思考，一直不能确定自己到底该有怎样的归属，现在忙的感觉终究是一场空。

说好了从四月份开始提出离职的申请，到现在还没兑现。也许是挣扎的太久，害怕现状的改变；也许又是那段时间面试几份工作失败后对自己能力的怀疑，于是便想趁着项目还没结束，赶紧补充一下自己的知识，从而赢得更好的工作机会。一段时间过去了，发现事情并不是这样，很多东西失去了意义，有时脑中便会冒出：“这他妈到底是什么？”。

六月的星空却在这两天绽放出了光彩，让人惊叹不已。生活中的美好往往也会不期而至，恰逢这样难得的天气和星空之前，我随姐夫一家游览了北京天文馆，意外的发现宇宙很多奇妙的东西，还观赏了一部惊奇的短片“奇妙的星空”。那一刻抑或两三秒中，我似乎看到童年时光的影子，因为影片中某个突如其来的周边景象唤起了童年的记忆，而童年又牵扯内心本原的太深太深……了解了春夏秋冬星空的变化，对星座有了初步认识，原来这一切都是这么的神奇！
