git常用命令

发表于 2016-01-01 | 分类于 技能

远程操作

  • 克隆某分支仓库
  • 添加远程主机(rhost)的仓库
  • 显示远程主机详细信息
  • 推送新分支dev到远程主机分支dev
  • 标签推送

本地操作

  • 新建分支dev并切换到该分支
  • 查看别人推送分支并与本地建立追踪
  • 重命名分支dev为develop
  • 提交时显示所有diff信息
  • 放弃当前正在合并分支
  • 删除分支
  • 先预览(加参数-n)，剔除版本控制
  • 查看文件的每一行是哪个提交最后修改的
  • 查看今天或截止6天前写了多少行代码

git log操作

  • 查看文件的每一个详细的历史修改，如果没有-p选项，只显示提交记录，不显示文件内容修改，git log -p -3 filename 显示最近的3次提交
  • 每一行显示一个提交，先显示哈希码，再显示提交说明
  • 查看自己某人最后提交的N条记录

git stash 用法

  • 备份当前的工作区的内容，从最近的一次提交中读取相关内容，让工作区保证和上次提交的内容一致。同时，将当前的工作区内容保存到Git栈中
  • 显示Git栈内的所有备份，可以利用这个列表来决定从那个地方恢复
  • 将工作应用回来
  • 从Git栈中读取最近一次保存的内容，恢复工作区的相关内容。由于可能存在多个Stash的内容，所以用栈来管理，pop会从最近的一个stash中读取内容并恢复
  • 指定版本号为stash@{1}的工作取出来
  • 清空Git栈
